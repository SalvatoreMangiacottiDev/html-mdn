What is HTML?

HTML (HyperText Markup Language) is a markup language that defines the content and structure of any website. It consists of a series of elements made up of tags and content. tags define the structure and functionality, while content is what gets displayed on the webpage.

Structure of an element

Opening tag → <p> Content </p> ← Closing tag

Nesting elements

<p> <strong> Content </strong> </p>

Void elements

Some elements consist of a single tag, primarily used to insert or embed content into a document. These elements called void elements, do not require a closing tag.

<img src="image.jpg" alt="Description">

Structure of an HTML document

                1. <!doctype html>

                2. <html lang="en-US">

                3. <head>

                    4. <meta charset="utf-8" />

                    5. <title> My webpage </title>

                </head>

                6. <body>

                    <p> This is my webpage </p>

                </body>

                </html>
  1. The <!DOCTYPE html> declaration specifies that the document follows HTML5 standards, ensuring proper browser interpretation and rendering.
  2. The <html lang="en-US"> element, known as the root element encloses all page content and defines the document's language for accessibility and search optimization.
  3. The <head> element contains metadata and resources that affect the page's behavior and appearance, without being directly visible to users.
  4. The <meta charset="utf-8" /> element specifies UTF-8 as the character encoding, ensuring correct display of text from various languages and preventing formatting issues.
  5. 5.The <title> element defines the page title, shown in the browser tab and used when bookmarking the page.
  6. The <body> element contains all visible content, including text, images, videos, and other elements displayed on the page.
